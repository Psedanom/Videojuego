# Accesibilidad para Dislexia

Para ejecutar el juego con las mejoras de accesibilidad para personas con dislexia, sigue los pasos a continuación:

1. Cambia la siguiente variable en el archivo `Videojuego/game_logic.js` en la línea 15:

```js
let font = "regular";
```

por:

```js
let font = "dyslexic";
```
