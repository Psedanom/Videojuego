Instrucciones para correr DeadDraw>
Se tiene que correr el archivo game_view.html 
se puede correr con live server o desde los archivo abriendolo desde el navegador
 
¿En qué escena comienza el prototipo?
Comienza con una pantalla de inicio (como un menu)
Seguiria un dialogo que le da un poco de xontexto narrativo al personaje
Por ultimo empezaria el juego principal

¿cuáles son los controles del juego?
"Espacio" para continuar en las pantallas de perdido,ganado y al inicio en el menu
"click izquierdo"para poder clickear las cartas y los botones, tambien para saltar los dialogos 
"p" para ganar la ronda (este control es para debugging)
Ademas la forma de jugar el juego esta en el archivo mecanicas.html

¿qué funcionalidad ya está terminada y qué aun está en desarrollo?
Funcionalidades terminadaas:
Las mecanicas principales del juego tales como que hace la carta arma enemigo y vida, Ademas de como dependiendo en que boton 
pones la carta es diferente como interactuan.
Un contador el cual si llega a cero el jugador pierde.
Dinero el cual se obtiene matando enemigos con el arma.
Vida del jugador funcional, con funcional nos referimos a que la cartas influyen en ella y ademas si esta misma llega a cero
se pierde por lo cual no es solo algo visual.
Obtencion de habilidades.
Interacciones de habilidades (las habilidades que tienen las cartas ya interactuan con otras tales como "enemieslos" que le quita -1 a el numero de las cartas enemigas).

Funcionalidades en desarrollo:
La tienda donde el jugador puede comprar la lootboxes.
Las mecanicas de los bosses.
Que el jugador pueda escoger el camino que quiere tomar.
El tutorial.
Habilidades extras de las cartas.
Boton para skipear una mano (son las cuatro cartas que estan en el tablero).
